{\footnotesize\ttfamily void GPSSim\+::loop (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [inline]}, {\ttfamily [private]}}



Loop principal responsável pela geração e transmissão de dados simulados. 

Esta função executa um laço contínuo enquanto o simulador estiver ativo ({\ttfamily \+\_\+is\+\_\+exec}). Em cada iteração\+:
\begin{DoxyItemize}
\item Inicializa ou atualiza a posição simulada (latitude e longitude).
\item Gera uma sentença NMEA do tipo GGA a partir da posição atual.
\item Transmite a sentença gerada através do descritor de escrita {\ttfamily \+\_\+fd\+\_\+pai}.
\item Aguarda o período de atualização definido em {\ttfamily \+\_\+periodo\+\_\+atualizacao}.
\item Atualiza a posição simulada chamando {\ttfamily \+\_\+update\+\_\+position()}.
\end{DoxyItemize}

O loop termina automaticamente quando {\ttfamily \+\_\+is\+\_\+exec} é definido como falso.

\begin{DoxyNote}{Note}
Esta função é bloqueante e deve ser executada em uma thread dedicada para não interromper o fluxo principal do programa. 
\end{DoxyNote}
