\doxysubsection{Detailed Description}
Representa um sensor GPS (GY-\/\+GPS6\+MV2 ou Simulado). 

Responsável por\+:
\begin{DoxyItemize}
\item Obter informações GNSS reais ou simuladas
\item Interpretar e armazenar esses dados
\item Enviar os dados via socket UDP em formato CSV
\end{DoxyItemize}

Para atender essas necessidades, foi implementada diversas funções independentes em thread e com formas inteligentes de organização.
