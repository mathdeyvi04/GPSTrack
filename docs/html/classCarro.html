Detailed Description

Representa um carro no sistema.

A classe Carro gerencia informações e comportamentos de um veículo, incluindo sua marca, modelo, velocidade atual e métodos para acelerar, frear e obter dados.

Exemplo de uso:

Carro c("Toyota", "Corolla");
c.acelerar(20);
std::cout << c.getVelocidade() << std::endl;
